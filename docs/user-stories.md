# User Stories

## Catálogo de Produtos
Como cliente
Quero visualizar produtos
Para escolher itens para compra

## Minha Conta
Como usuário autenticado
Quero acessar meus dados
Para gerenciar minha conta

## Meus Pedidos
Como cliente
Quero visualizar meus pedidos
Para acompanhar entregas

## Endereços
Como cliente
Quero cadastrar endereços
Para facilitar futuras compras

## Detalhes da Conta
Como usuário
Quero editar meus dados
Para manter cadastro atualizado
